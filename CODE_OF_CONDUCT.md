# Code of Conduct

KVFabric is a student-led open source project. We want the repository to stay collaborative, rigorous, and kind.

## Our expectations

- Be respectful in technical disagreement.
- Critique ideas, not people.
- Prefer concrete evidence over dismissive claims.
- Share context when proposing architectural changes.
- Assume good intent and help others get unblocked.

## Unacceptable behavior

- Personal attacks or harassment
- Repeated bad-faith arguments
- Doxxing or sharing private information
- Derailing technical discussions with hostility

## Scope

This code of conduct applies to issues, pull requests, discussions, and project communication channels.
