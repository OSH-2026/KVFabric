1. vLLM 的标准基础服务性能

vLLM benchmark docs: https://docs.vllm.ai/en/stable/performance/benchmarks.html
(新入口：https://docs.vllm.ai/en/stable/contributing/benchmarks.html)

vLLM benchmark README: https://github.com/vllm-project/vllm/blob/main/benchmarks/README.md

inference-perf: https://github.com/kubernetes-sigs/inference-perf

Vidur paper: https://www.microsoft.com/en-us/research/uploads/prod/2024/05/vidur_mlsys24.pdf

主要内容：

跑通 vllm bench serve、vllm bench throughput 等基础测试。
记录不同输入长度、输出长度、并发数下的性能变化。
关注 TTFT、TPOT、E2E latency、throughput、GPU memory 等指标。

需要解决：

① vLLM 在普通 serving 场景下性能是多少

② 系统瓶颈主要在 prefill、decode、调度还是显存

③ 后续 KVCache 优化主要提升什么，是降低首 token 延迟，还是提升吞吐

产出任务：

baseline_benchmark_method 报告
一组可运行的 vLLM benchmark 命令
指标解释表和基础结果表


2. KVCache 复用与前缀缓存评测

资料：

LMCache paper: https://arxiv.org/abs/2510.09665

LMCache GitHub: https://github.com/LMCache/LMCache

CacheBlend: https://blog.lmcache.ai/en/2025/03/31/cacheblend-best-paper-acm-eurosys25-enabling-100-kv-cache-hit-rate-in-rag/

Marconi paper: https://arxiv.org/abs/2411.19379

Marconi artifact: https://github.com/ruipeterpan/marconi

主要内容：

研究多个请求之间如何复用 KVCache
重点看共享 system prompt、RAG 文档复用、多轮对话复用等场景
分析 KVCache 命中后能减少多少 prefill 计算，以及 TTFT 能下降多少

需要解决：

① 什么样的请求可以复用 KVCache

② KVCache 命中率、prefix hit rate、saved prefill tokens 怎么定义

③ 缓存查找、加载、迁移本身会不会带来额外开销

④ 这些方法中哪些适合用于 KVFabric

产出任务：

kvcache_reuse_benchmark_method 报告
KVCache 复用指标表
共享 system prompt、RAG、多轮对话三类测试场景
LMCache、CacheBlend、Marconi 与 KVFabric 的对比分析


3. KVCache 压缩与质量评测

资料：

KVFundaBench: https://arxiv.org/abs/2502.01941

Stateful KV Cache Management: https://arxiv.org/abs/2511.04686

TurboQuant: https://openreview.net/pdf/6593f484501e295cdbe7efcbc46d7f20fc7e741f.pdf

KVFlow: https://arxiv.org/abs/2507.07400

TokenDance: https://arxiv.org/abs/2604.03143

主要内容：

研究 KVCache 被压缩、裁剪或量化后，模型质量是否下降
重点看长上下文问答、多轮对话、代码生成、推理任务等场景
同时记录显存节省、速度提升和质量损失

需要解决：

① KVCache 优化后模型输出是否变差

② 哪些任务最容易暴露质量下降

③ 压缩率、显存节省、吞吐提升和准确率下降之间怎么比较

④ KVFabric 当前阶段是否需要纳入这类质量评测

产出任务：

kvcache_quality_benchmark_method 报告
质量评测任务清单
性能收益与质量损失对比表
对 KVFabric 后续质量评测的建议
